# MyDeltas
>数据变化,类似OData中的Delta

## 三个功能项目
### 1、[MyDeltas](./deltas.md)
>* 主项目
>* 数据变化,类似OData中的Delta
>* 基于反射的PropertyInfo和FieldInfo

### 2、[MyDeltas.Emit](./emit.md)
>* 提供emit代替PropertyInfo和FieldInfo来操作数据变化

### 3、[MyDeltas.OpenApi](./openapi.md)
>* ApiSchema转化器
>* 让OpenApi文档的MyDelta参数与实际效果一致
